For-loop
{% for i in 1..10 %}
    <div class="{{ cycle(['even', 'odd'], i) }}">
      <!-- some HTML here -->
    </div>
{% endfor %}

Renderanje templatea:
1. Definiram base layout

Bitno:
{% block sidebar %}{% endblock %} - Blok koji se može overridati ili se može prikazat neki defaultni unutar ovih tagova

2. Definiram child templatea

Bitno:
{% extends 'base.html.twig' %} - Prvo rendera base, pa onda ovaj template overrida ostalo; mora biti prvi tag u templateu
Ako želimo overridano dodati s default, koristimo {{ parent() }}

Svi templatei su relativni s templates/ folderom.
Template može biti i u bundleu. Onda se poziva s npr. @AcmeBlog/Blog/index.html.twig

{{ include('template', 'arguments') }} - Printa template koji nije u tome
{{ path('route') }} - Za stvaranje linkova bez hardcodanja

U kontroleru:
    /**
     * @Route("/", name="welcome")
     */

U PHP-u (config/routes.php):
$routes = new RouteCollection();
$routes->add('welcome', new Route('/', array(
    '_controller' => 'App\Controller\WelcomeController::index',
)));

return $routes;
